# Plans for the app

## Pages

- Home Page
- Ranking Page
- Stats Page
- Setting Page

## Page Functionality

- Home Page:
  Empty for now

- Ranking Page:
  Display the ranking of the weather stations
    - Ranking:
      - Based on how long a weather station held a gold star
      - Total amount of days with gold star

- Stats/History Page
  Display the long gold star streak for a week, month and year

- Setting Page
  Allow to change theme
